# 🎬 Netflix Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing Netflix's content library using Python to uncover trends and patterns in movies and TV shows. The analysis involves data cleaning, exploratory data analysis (EDA), and data visualization to gain insights into content distribution, ratings, genres, countries, and release trends.

---

## 🎯 Business Problem

With thousands of titles available on Netflix, understanding content distribution and viewing trends is essential for content planning and business strategy. This project aims to answer key questions such as:

- What type of content dominates Netflix?
- Which countries produce the most Netflix content?
- What are the most common content ratings?
- How has Netflix's content library grown over the years?
- Which genres are most popular?

## 🧹 Data Preparation

- Imported and explored the dataset using Pandas.
- Checked data types and converted required columns to appropriate formats.
- Handled missing values in columns such as **Director, Cast, Country, and Rating**.
- Removed duplicate records.
- Converted **Release_Date** to datetime format for time-based analysis.
- Prepared the dataset for exploratory data analysis and visualization.

---

## 📌 Key Insights

- Movies account for a larger share of Netflix's content compared to TV Shows.
- **TV-MA** and **TV-14** are the most common content ratings, indicating a strong focus on mature audiences.
- The **United States** contributes the highest number of Netflix titles.
- Netflix experienced significant growth in content additions after **2015**.
- Drama and International Movies are among the most frequently available genres.

---

## 💼 Business Value

The analysis provides valuable insights that can help streaming platforms:

- Understand audience preferences based on content ratings.
- Identify high-content-producing countries for strategic partnerships.
- Analyze content growth trends over time.
- Support content acquisition and production decisions.
- Improve regional content planning based on market demand.
